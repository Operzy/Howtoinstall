Install Visual Studio Code

STEP 02
Install Node.js
DOWNLOAD NODE.JS → nodejs.org
1 Click "Get Node.js" and select your operating system (it should auto-detect).
2 Download the installer and install it like any normal app.
GOOD TO KNOW
Node.js isn't strictly required for Claude Code to launch — but you'll want it installed. Most of the tools, frameworks, and packages Claude works with run on Node under the hood. You don't need to know JavaScript — just think of it as the engine that keeps everything running smoothly behind the scenes.
Node.js is installed
STEP 03
Install Python
DOWNLOAD PYTHON → python.org/downloads
1 Click on Downloads and download the version for your operating system.
2 Follow the prompts and install like any normal app.
GOOD TO KNOW
Same idea as Node.js — Python isn't a hard requirement to get Claude Code running, but it powers a huge portion of the AI and automation ecosystem. When Claude generates scripts, processes data, or works with APIs, Python is often what it reaches for. Install it now so you're ready when it comes up.
Python is installed
STEP 04
Install the Claude Code Extension

OPEN VS CODE

1 Open VS Code. If it's your first time, you'll see an onboarding screen — just skip it or mark it as done.
⚠ HEADS UP
You'll see a built-in AI chat on the right side of VS Code. This is NOT Claude Code — it's Microsoft's default assistant. Ignore it for now.

FIND & INSTALL

2 On the left sidebar, click the four squares icon (it looks like building blocks). This opens the Extensions Marketplace.
3 Search for "Claude Code". You'll see multiple results — look for the one by Anthropic with a blue verification tick next to the name. That's the official one.
4 Click Install. When it asks if you trust the publisher, click Yes.
Claude Code extension is installed

SET UP YOUR VIEW

5 A "Getting Started" screen will pop up — just close it. Then close the Extensions panel by clicking the same four-squares icon again.
6 You should now see two new things:
■ A Claude icon on the left sidebar
■ A Claude Code tab in the right-side chat panel
Click the Claude Code tab on the right — that's where you'll chat with Claude.
CLEAN UP
To get rid of the default Microsoft chat: right-click it and select "Hide Chat". This keeps Claude Code front and center whenever you open VS Code.
STEP 05
Connect Your Claude Account
⚠ PAID FEATURE
Claude Code requires at minimum the Pro plan ($20/mo monthly, $17/mo annual). Also available on Team and Enterprise plans. Higher plans (Max 5x, Max 20x) give more usage before hitting limits. On Pro, you'll eventually hit a usage limit and need to wait a few hours.
1 In the Claude Code panel, select the login option "Claude AI Subscription".
2 It will prompt you to open a website — click Open.
3 If you are already logged into your Claude account, select your organization.
4 Click Authorize to give Claude Code access to your Claude account.
5 A confirmation page will say "Build something great — you're all set". Close that browser window.
DONE
Claude Code is now connected to your account and ready to use in VS Code!

STEP 06
One Final Step Before You Prompt
⚠ IMPORTANT
Do NOT start prompting Claude Code immediately. The first thing you must always do is open a dedicated folder to work in.
1 Go to File → Open Folder (or use the Open option if visible).
2 If starting a new project, create a new folder for it. We recommend having a dedicated folder for all your Claude Code projects.
3 VS Code will ask if you trust the authors of the files — since you're starting fresh, click "Yes, I trust the authors".
4 When VS Code opens the folder, it may switch back to the default chat — make sure you click back on the Claude Code tab before starting.
